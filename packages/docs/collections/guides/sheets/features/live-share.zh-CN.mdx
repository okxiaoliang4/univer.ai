# Live Share

| 有付费版本 | 需要接入 Univer 服务端 | 可用于 Univer on Node.js | 包含的 Preset |
| - | - | - | - |
| ✅ | ✅ | - | `UniverSheetsCollaborationPreset` |

Live Share 是基于协同编辑提供的实时演示功能，**因此在使用 Live Share 之前，需要先安装 [协同编辑](/guides/sheets/features/collaboration) 插件**。

## 高级安装

<Tabs items={['pnpm', 'npm']}>
  <Tabs.Tab>
    ```shell
    pnpm add @univerjs-pro/live-share
    ```

```typescript
import { UniverLiveSharePlugin } from '@univerjs-pro/live-share';
import '@univerjs-pro/live-share/lib/index.css';

univer.registerPlugin(UniverLiveSharePlugin);
```
