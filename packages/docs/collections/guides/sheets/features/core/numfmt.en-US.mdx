# Number Format

| 有付费版本 | 需要接入服务端 | 可用于 Univer on Node.js | 包含的 Preset |
| - | - | - | - |
| - | - | ✅ | ❌ 通过 plugin 引入 |

数字格式插件提供了对单元格中数字的格式化功能，包括数字的小数位数、千分位分隔符、货币符号等。

该功能包含以下插件包：

* [`@univerjs/sheets-numfmt`](https://www.npmjs.com/package/@univerjs/sheets-numfmt) - 数字格式插件

<Callout type="warning" emoji="🚨">
  不支持 **DBNum** 语法
</Callout>

```typescript {2-3,5,7,14,19}
import { LocaleType, Tools } from '@univerjs/core';
import { UniverSheetsNumfmtPlugin } from '@univerjs/sheets-numfmt';
import SheetsNumfmtZhCN from '@univerjs/sheets-numfmt/locale/zh-CN';

import '@univerjs/sheets-numfmt/lib/index.css';

import '@univerjs/sheets-numfmt/facade';

const univer = new Univer({
  theme: defaultTheme,
  locale: LocaleType.ZH_CN,
  locales: {
    [LocaleType.ZH_CN]: Tools.deepMerge(
      SheetsNumfmtZhCN
    ),
  },
});

univer.registerPlugin(UniverSheetsNumfmtPlugin);
```
## Facade API

### 设置数字格式

```typescript
const sheet = univerAPI.getActiveWorkbook().getActiveSheet();

// get range
const range = sheet.getRange(0, 0, 10, 10); // A1:J10

// 具体参数请参考 https://support.microsoft.com/en-us/office/number-format-codes-5026bbd6-04bc-48cd-bf33-80f18b4eae68
range.setNumberFormat('#,###');
```
