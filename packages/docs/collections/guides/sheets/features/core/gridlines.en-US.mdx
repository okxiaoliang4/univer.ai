# Gridlines <VersionBadge version="0.4.2+" />

<FeatureMeta texts={['-', '-', '-', 'UniverSheetsCorePreset' ]} />

From version 0.4.2, you can toggle the visiblity of a Worksheet's gridlines.

## Facade API

### Show or line gridlines of a worksheet

```typescript
import { FUniver } from '@univerjs/facade';

const univerAPI = FUniver.newAPI(univer);
const sheet = univerAPI.getActiveWorkbook().getActiveSheet();

// hide
sheet.setHiddenGridlines(true); 

// show
sheet.setHiddenGridlines(false);
```

### Get if the sheet's gridlines or hidden

```typescript
import { FUniver } from '@univerjs/facade';

const univerAPI = FUniver.newAPI(univer);
const sheet = univerAPI.getActiveWorkbook().getActiveSheet();

const hidden = sheet.hasHiddenGridlines();
```
