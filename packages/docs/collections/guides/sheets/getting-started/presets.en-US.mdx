# Presets  <VersionBadge version="0.5.0+" />

From version 0.5.0, Univer provides the `@univerjs/presets` package, which provides presets for common scenarios to help you quickly build a Univer application without worrying about how to import and configure Univer plugins.

<Callout type="info" emoji="ðŸ›¸">
  If you are already familiar with Univer and want to import plugins more flexibly (e.g., lazy loading some plugins, and using some configuration options not provided by Presets), you can refer to [Advanced Installation](/guides/sheets/getting-started/advanced-installation).
</Callout>


## Basic Spreadsheet

Create a simple spreadsheet and reference all the first-party features (plugins) provided by Univer.

```js
import { createUniver, defaultTheme, LocaleType } from '@univerjs/presets';
import { UniverSheetsBasicPreset } from '@univerjs/presets/sheets/sheets-basic';
import { enUS } from '@univerjs/presets/sheets/sheets-basic/en-US';

const { univerAPI } = createUniver({
    locale: LocaleType.EN_US,
    theme: defaultTheme,
    presets: [
        UniverSheetsBasicPreset({ locales: { enUS } }),
    ],
});

univerAPI.createUniverSheet({ name: 'Test Sheet' });
```

## Spreadsheet with Advanced Features

To use the advanced features provided by Univer (including printing, pivot tables, etc.), you need to import `UniverSheetsAdvancedPreset`.

1. If you have obtained authorization, you need to configure the authorization information.
2. If you have the server setup, you need to configure the server endpoint.

```js
import { createUniver, defaultTheme, LocaleType } from '@univerjs/presets';
import { UniverSheetsAdvancedPreset } from '@univerjs/presets/sheets/sheets-advanced-features';
import { enUS as advancedEnUS } from '@univerjs/presets/sheets/sheets-advanced-features/en-US';
import { UniverSheetsBasicPreset } from '@univerjs/presets/sheets/sheets-basic';
import { enUS as basicEnUS } from '@univerjs/presets/sheets/sheets-basic/en-US';

const { univerAPI } = createUniver({
    locale: LocaleType.EN_US,
    theme: defaultTheme,
    presets: [
        UniverSheetsBasicPreset({ locales: { enUS: basicEnUS } }),
        UniverSheetsAdvancedPreset({ 
            locales: { enUS: advancedEnUS }, 
            universerEndpoint: 'https://dev.univer.plus',
            license: 'xxx',
        }),
    ],
});

univerAPI.createUniverSheet({ name: 'Test Sheet' });
```

Online Example.

## Spreadsheet with Collaboration

```js
import { createUniver, defaultTheme, LocaleType } from '@univerjs/presets';
import { UniverSheetsAdvancedPreset } from '@univerjs/presets/sheets/sheets-advanced-features';
import { enUS as advancedEnUS } from '@univerjs/presets/sheets/sheets-advanced-features/en-US';
import { UniverSheetsBasicPreset } from '@univerjs/presets/sheets/sheets-basic';
import { enUS as basicEnUS } from '@univerjs/presets/sheets/sheets-basic/en-US';
import { UniverSheetsCollaborationPreset } from '@univerjs/presets/sheets/sheets-collaboration';
import { enUS as collaborationEnUS } from '@univerjs/presets/sheets/sheets-collaboration/en-US';

createUniver({
    locale: LocaleType.EN_US,
    theme: defaultTheme,
    collaboration: true,
    presets: [
        UniverSheetsBasicPreset({ locales: { enUS: basicEnUS }, collaboration: true }),
        UniverSheetsAdvancedPreset({ locales: { enUS: advancedEnUS } }),
        UniverSheetsCollaborationPreset({ locales: { enUS: collaborationEnUS } }),
    ],
});
```

Online Example.
