# Univer Sheets

欢迎使用 Univer Sheet 文档！

## Univer Sheet 是什么

Univer 是一个用于构建生产力应用程序的全栈框架，Univer Sheet 是 Univer 的电子表格组件。

Univer Sheet 可以在浏览器和服务器（Node.js）上运行，并且拥有灵活、可扩展和可定制的特性，你可以用它构建各种应用程序，无论是简单的计算器还是企业级的复杂财务模型。无论你是个人开发者还是团队的一员，Univer Sheet 都可以帮助你快速轻松地构建强大的基于电子表格的应用程序。

## 主要功能

Univer Sheet 现在支持广泛的功能，包括：

- **基本电子表格功能**：创建、编辑和删除单元格、行和列。
- **公式**：使用公式计算单元格中的值。
- **数字格式**
- **数据验证和条件格式**
- **筛选和排序**
- **权限控制**
- **浮动图片**
- 以及更多...

请参阅 [功能](/guides/sheets/features/core) 部分下的页面以了解详情。核心和主要功能在 [GitHub 上开源](https://github.com/dream-num/univer)并且支持免费使用。

## 如何使用这些文档

在屏幕的左侧是文档导航栏。文档页面按从基础到高级的顺序组织，你可以按照这个顺序循序渐进地了解 Univer，或者跳到你感兴趣的章节。你还可以在侧边栏的顶部切换 Univer Docs 的文档。

## 前置知识

尽管我们已经尽力使得文档对新手也易于理解，但是我们必须假设你在开始阅读之前已经了解了一些基础知识，以便我们的文档可以专注于 Univer。

为了充分利用我们的文档，建议你对 JavaScript 及其生态系统有基本的了解。

如果你有兴趣在 Node.js 上运行 Univer，你还应该对 Node.js 有基本的了解。

如果你想尝试高级功能并部署 Univer Server，我们建议你对 Linux、Docker 和 Docker Compose（或 Kubernetes）有基本的了解。

## 了解高级功能

我们为企业用户提供一系列专业功能，包括：

- **数据透视表**
- **打印**
- **导入和导出**
- **协作和共享**
- 以及更多...

这些功能在文档中将用 <span className="mr-2"><TitleTag /></span> 标记。

高级功能基于 Univer 商业许可协议，可以免费使用，但有一些限制。你可以查看[高级功能介绍](/guides/sheets/pro-features/introduction)了解更多详情。

## 扩展 Univer Sheet

在设计之初，Univer 就被设计为高度可扩展的，因此你可以根据自己的需求来定制它。我们建议你阅读[架构](/introduction/architecture/univer)部分，了解 Univer Sheet 的结构以及如何扩展它。也可以从 [如何编写 CSV 导入插件](http://localhost:3000/en-US/guides/sheets/tutorials/csv-import-plugin) 开始。
